□========================================
□ 作業記録補助ソフト WorkRec Ver 1.1
□========================================
【このソフトについて】
一定時間おきに記録ウィンドウを表示して、作業記録をつける為のソフトです。
自動では記録しませんので、「記録をつけるぞ！！」と言う強い意志が必要です。

【動作環境】
Windows XP 以降
.NET Framework 3.5

【使い方など】
1.起動するとタスクトレイにアイコンが表示されます。

2.設定時間、デフォルトで１５分が経過すると、記録ウィンドウが表示されますので
  作業内容を入力して記録ボタンをクリックします。

3.ダイアログを閉じてから設定時間が経過すると、また記録ウィンドウが表示されます。
  以降、繰り返し。

○入力した内容は最近５件の履歴を持ち、ラジオボタンをクリックすると
  その内容で記録されます。

○記録ウィンドウのショートカットキー
  記録ボタン      CTRL + ENTER
  ラジオボタン    CTRL + 1 〜 5

【インストール】
適当なフォルダに解凍してください。

.NET Framework 3.5 がインストールされていない時は
マイクロソフトのサイトよりダウンロードして、インストールして下さい。
http://msdn.microsoft.com/ja-jp/netframework

【アンインストール】
解凍したファイルを削除して下さい。

設定ファイルはユーザフォルダの中の \AppData\Local\WorkRec\ にあります。

【開発環境】
Windows7 32bit Ultimate
Visual C# 2008 Express Edition
